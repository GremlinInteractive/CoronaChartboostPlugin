Overview

Initializes the Chartboost library. This function is required and must be executed before making other Chartboost calls such as chartboost.show().

Syntax

````` chartboost.init( options ) `````

This function takes a single argument, options, which is a table that accepts the following parameters:

licenseKey - (required)

String. Your license key for the Chartboost plugin. This can be purchased from the Gremlin Interactive Store. If a license key is invalid or not provided, the Chartboost plugin will show a license reminder popup and will not function. Therefore you must purchase a valid license to use this plugin.

testMode - (optional)

Boolean. If you're ready to use Chartboost in a live/published app, set this value to false. If you wish to just test this plugin or you're still in the development phase, set this value to true. Default is true.

appID - (required)

String. Your Chartboost app ID. You can get your app ID from the Chartboost website.

appSignature - (required)

String. Your Chartboost app signature. You can get your app signature from the Chartboost website.

listener - (optional)

Listener. This function is executed after your license key has been validated. It's recommended that you listen for this event in case of network errors. This function returns event.status with possible values of valid, invalid, or networkError. This function also receives Chartboost events. With event.type of "chartboost". The events returned for Chartboost are as follows: "interstitial" Phases of: "willDisplay", "didDisplay", "closed", "clicked", "cached", "load". "moreApps" Phases of: "willDisplay", "didDisplay", "closed", "clicked", "cached", "load".

Example

 -- Require the Chartboost library local chartboost = require( "plugin.chartboost" ) -- Initialize the Chartboost library chartboost.init( { licenseKey = "Insert_Your_License_Key_Here", testMode = true, appID = "app_ID_generated_from_chartboost_here", appSignature = "app_signature_generated_from_chartboost_here", listener = function( event ) -- Print the events key/pair values for k,v in pairs( event ) do print( k, ":", v ) end end })
